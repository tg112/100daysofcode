##### string reverse

使用したメソッド

- `split()`: 文字列を配列にするメソッド
- `reverse()`: 配列の順番を逆にするメソッド
- `join()`: 配列を文字列にするメソッド
- `reduce`: 第二引数が初期値で、初期値に対して第一引数の値を処理する。

なるべく `for(let i = 0; i < a.length; i++){}`の形は避ける。`for of`や`for in`を使用して複雑性を避ける
